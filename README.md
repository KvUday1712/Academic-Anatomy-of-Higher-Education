# Academic Anatomy Of Higher Education

## Overview

**Academic Anatomy of Higher Education** is a centralized academic management platform designed to simplify and automate key academic and administrative processes in higher educational institutions.  
The system provides structured, role-based access for **Administrators, Staff, and Students**, ensuring efficient data management, transparency, and smooth academic operations.

The platform focuses on managing **student and staff records**, **attendance**, **results**, **academic references**, and **leave requests** within a single integrated system.

---

## Objectives

- Centralize academic and administrative data
- Reduce manual work and paperwork
- Improve accuracy in attendance and result management
- Provide easy access to academic reference materials
- Enable a structured leave request and approval process

---

## User Roles and Functionalities

The system consists of three main user roles:

---

### Admin Module

The **Admin** has complete control over the platform and manages the overall academic structure.

**Features:**
- Add, update, and manage staff records
- Add, update, and manage student records
- Create and manage courses
- Create and manage subjects
- View, approve, or reject student leave requests

---

### Staff Module

The **Staff** module is responsible for academic evaluation and content management.

**Features:**
- Mark and manage student attendance
- Add and update student results
- Upload and manage academic reference materials

---

### Student Module

The **Student** module allows students to access academic information and submit requests.

**Features:**
- View attendance records
- View results and marks
- Access reference materials uploaded by staff
- Submit leave requests to the admin
- Track leave request approval status

---

## Key Features

- Role-based authentication and authorization
- Centralized academic management system
- Attendance and result tracking
- Reference material management
- Leave request and approval workflow
- Scalable and user-friendly design

## Installation and Setup

1. Clone the repository
   ```bash
   git clone https://github.com/KvUday1712/Academic-Anatomy-Of-Higher-Education.git
   ```
2. Navigate to the project directory
   ```bash
   cd Academic-Anatomy-Of-Higher-Education
   ```
3. Install required dependencies
  ```bash
  pip install -r requirements.txt
  ```
4. Run the application
  ```bash
  python app.py
  ```
5. Open your browser and navigate to
```bash
http://localhost:5000
```
